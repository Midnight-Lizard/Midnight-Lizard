Midnight Lizard cannot connect to this page

because of one of the following reasons
  • Web browser does not allow to run extensions on this page
    (e.g. Web Store, Settings, New Tab)

  • [Midnight Lizard] has been updated or just installed
    so page needs to be refreshed

  • Current page represented by a local file,
    but you have not allowed [Midnight Lizard] to process local files

CAUTION
Midnight Lizard is extension so it cannot change colors of browser tabs and toolbars. For this purpose you need to install/change a custom browser theme.
